The Crossref Open Citation Identifier (COCI)

The Open Citation Identifier (OCI) is a globally unique persistent identifier for bibliographic citations, created and maintained by OpenCitations, this project takes in consideration a large number of bibliographic resources retrieved from Crossref (COCI) and annotates/elaborates their citing information.

Each COCI has a simple structure: the lower-case letters “ci” followed by a colon, followed by two numbers separated by a dash. The first number identifies the citing resource, while the second one identifies the cited resource. The Crossref bibliographic resource identifiers have a supplier prefix equl to "020". For example, ci/02001010806360107050663080702026306630509-02001010806360107050663080702026305630301 is a valid COCI.

You can work and iteract with the COCI triplestore using the following services/applications:

  • A triplestore searching interface (built using OSCAR)

  • A resource browser (built using LUCINDA). In order to use this service, a resource identifier must be specified in the URL address.
    For instance: /index/coci/browser/ci/02001010806360107050663080702026306630509-02001010...

  • A SPARQL editor to query the COCI triplestore
