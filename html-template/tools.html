Tools

OpenCitations provides additional tools for retrieving information about bibliographic and citation data. In particular:

  • Open Citation Identifier Resolution Service. OpenCitations provides an resolution service for Open Citation Identifiers (OCIs). An OCI is a globally unique persistent identifier for bibliographic citations, created and maintained by OpenCitations. The resolution service enables one to specify an OCI and returns information about that citation.
